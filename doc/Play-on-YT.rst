
Play on YouTube
+++++++++++++++

Through ``playonyt`` function you can play video on YouTube

Code
====

.. code-block:: python

   import pyreskit

   pyreskit.playonyt("Rick Roll")

OR

.. code-block:: python

   from pyreskit import playonyt

   playonyt("Rick Roll")
